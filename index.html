HTML - Estructura Básica

HTML es el lenguaje de marcado utilizado para crear páginas web. Escribe una estructura básica de HTML, con etiquetas como <html>, <head>, <body> y <h1>.

Ejemplo:

<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Mi Página Web</title>
    </head>
    <body>
        <h1>¡Bienvenidos al Mundo de la Programación!</h1>
    </body>
</html>

Ejercicio:

Escribe un código HTML que contenga un título utilizando la etiqueta <h1> y el texto: "¡Bienvenidos al Mundo de la Programación!"

CSS - Estilizando tu Página

CSS se utiliza para darle estilo a tus páginas web. Puedes cambiar colores, fuentes y disposición de los elementos.

Ejemplo:

body {
    background-color: #f4f4f4;
    color: #333;
}

Explicación:

Este código CSS cambia el color de fondo de la página a un gris claro y el color del texto a gris oscuro.

Ejercicio:

Usa CSS para cambiar el color de fondo de la página a un color claro y el color del texto a oscuro.

JavaScript - Interactividad

JavaScript permite añadir interactividad a las páginas web. Puedes hacer que el navegador ejecute código cuando el usuario interactúa con la página.

Ejemplo:

console.log("¡Hola Mundo!");

Explicación:

Este código JavaScript imprime el mensaje "¡Hola Mundo!" en la consola del navegador.

Ejercicio:

Escribe un código JavaScript que imprima "¡Hola Mundo!" en la consola del navegador.
